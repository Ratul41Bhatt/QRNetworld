Bank Integration Webhook Documentation

1. Webhook Payment Callback

To broadcast payments from your bank core network to registered Soundbox devices, compile a REST API webhook payload and execute an HTTP `POST` request to our gateway route. The gateway automatically translates the Terminal ID (TID), resolves the device's unique topic, and publishes the speech payload via MQTT.

POST https://your-soundbox-gateway.com/api/payment

Payload Parameters (JSON Body)

Parameter Type Status Description
tid String Required Terminal ID registered inside the database mapping console.
amount Number / String Required Payment transaction amount. Read aloud by the device speaker.
invoice String Optional Transaction invoice index (e.g. "459039"). Displays on device LCD screen.
rrn String Optional Retrieval Reference Number (RRN) for transaction audits. Displays on device screen.
card_number String Optional Masked account or card index triggering payment. Logged inside console metrics.

Integration Code Samples

cURL HTTP REQUEST 
curl -X POST https://your-soundbox-gateway.com/api/payment \
  -H "Content-Type: application/json" \
  -d '{
    "tid": "102000000105664",
    "amount": 250.50,
    "invoice": "402940",
    "rrn": "392040590390",
    "card_number": "1234-xxxx-xxxx-9012"
  }'
JAVASCRIPT (FETCH API) 
const payload = {
  tid: "102000000105664",
  amount: 250.50,
  invoice: "402940",
  rrn: "392040590390",
  card_number: "1234-xxxx-xxxx-9012"
};

fetch('https://your-soundbox-gateway.com/api/payment', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(payload)
})
.then(res => res.json())
.then(data => console.log('Speech triggered successfully:', data))
.catch(err => console.error('Connection failed:', err));
PYTHON (REQUESTS) 
import requests

url = "https://your-soundbox-gateway.com/api/payment"
payload = {
    "tid": "102000000105664",
    "amount": 250.50,
    "invoice": "402940",
    "rrn": "392040590390",
    "card_number": "1234-xxxx-xxxx-9012"
}

headers = {"Content-Type": "application/json"}
response = requests.post(url, json=payload, headers=headers)
print("Response Status Code:", response.status_code)
print("Response JSON:", response.json())
